Projetos

Alura Studies

Alura Studies é uma aplicação web desenvolvida em React, utilizando TypeScript e frameworks de CSS como SCSS.
Com Alura Studies, os usuários podem definir tópicos de estudo, estabelecer o tempo de estudo para cada tópico e usar um cronômetro integrado para controlar seu progresso.
Esta página permite que os estudantes maximizem sua produtividade e mantenham o foco em seus objetivos de aprendizado.
